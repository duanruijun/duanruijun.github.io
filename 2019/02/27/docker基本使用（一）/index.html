docker基本使用

  1. docker pull下载镜像

    下载一个nginx镜像，命令：docker image pull或者docker pull

    注意： 版本中带有alpine字样的是基本最小单元的版本，比如：一个完整的nginx镜像资源为50M左右大小，而alping则仅为10M左右，alpine版本一般用于测试使用，在生产环境中不建议使用。
